# NeuralPanic Creative Brief

## Hook
A voice artifact processor that sounds like a real-time AI call system losing coherence under stress.

## Description
NeuralPanic is a horror-focused speech mangler built around instability, not simple distortion.  
It emulates neural codec collapse, packet loss concealment errors, and vocoder identity drift.  
The sonic target is "something intelligent but failing": words stutter, formants slide, and tone changes unpredictably in short bursts.

## Sonic Character
- Warmth: Low
- Clarity: Variable (intentionally unstable)
- Character: Corrupted / synthetic / uncanny
- Primary sensation: fragile speech identity breaking in real time

## Use Cases
1. Horror voice design for trailers, games, and cinematic post.
2. Sci-fi comms breakdowns and emergency broadcast FX.
3. Glitch vocals and transitions in experimental music.
4. Live performance destruction for spoken-word sources.

## Core Design Principles
1. Instability over static coloration.
2. Real-time safe stochastic behavior (deterministic seedable randomness).
3. Macro-first workflow: one control can move many internals.
4. Harshness managed by output safety, not by reducing chaos.

## UI Direction
- Theme: emergency broadcast console with biomedical monitor aesthetics.
- Visual language: industrial labels, alarm states, live fault telemetry.
- Mood: clinical panic rather than neon cyberpunk.

## References
- Broken VoIP / voice chat under bad connection.
- Neural codec bitrate drop behavior.
- Packet-loss microloop artifacts.
- Broadcast limiter panic pumping.
