# Style Guide v1: Osmium

## Color Palette
- **Background Base:** `#121214` (Deep matte black/grey)
- **Surface:** `#1c1c1f` (Slightly lighter, card-like)
- **Primary Accent (Low):** `#00f3ff` (Cyber Cyan)
- **Primary Accent (High):** `#ff3300` (Reactor Orange)
- **Text Main:** `#ffffff`
- **Text Muted:** `#666666`

## Typography
- **Font Family:** 'Inter', system-ui, sans-serif (Clean, modern)
- **Headers:** Uppercase, widely spaced (letter-spacing: 2px)
- **Value Text:** Monospace, distinct.

## Visual Effects
- **The Core:**
    - Base: Dark metallic circle.
    - Active: `box-shadow: 0 0 20px var(--glow-color)`
    - Gradient: Conic gradient indicating value.
- **Surfaces:** Subtle 1px borders, rounded corners (4px).

## Interaction
- **Hover:** Brightness boost (+10%).
- **Active/Drag:** Cursor disappears, value readout highlights.
